## Abstract: 

What if you owned your social media data and could move freely between apps? ATProto makes this possible. I'll teach the fundamentals—DIDs, data repos, why they're important, and how to build your first decentralized social platform.

### 2. What is ATProto
ATProto (The Atmosphere)
ATProtocol is an open protocol, where you as the user own your own data. It moves your posts, likes, follows and other data into your own posession.
https://atproto.com
### 3. How ATProto works
- Each user gets their own repo with their data storage. Each user has an ID called a DID - Decentralized Identifier
- The user has a handle
- Handles are DNS records, similar to websites

Records and blobs are stored in the user's repo.
Handles can be found at (PDSLS.dev as well)[https://pdsls.dev/]
